## Overview
A hands-on home lab project demonstrating Active Directory administration and help desk support capabilities. This project showcases my ability to deploy, configure, and manage enterprise IT infrastructure.

## Project Goals
- Deploy a functional Active Directory domain controller
- Practice enterprise networking and security configurations
- Develop skills relevant to help desk and IT support roles
- Document technical procedures clearly for future reference

## Technologies Used
- **Virtualization**: Oracle VirtualBox
- **Operating System**: Windows Server 2022 Standard Evaluation
- **Services**: Active Directory Domain Services (AD DS), DNS
- **Networking**: Static IP configuration, TCP/IP fundamentals

## Lab Environment
- **Domain**: homelab.local
- **Domain Controller**: DC01-Server2022 (192.168.1.10)
- **Functional Level**: Windows Server 2016

## What I Learned
This project deepened my understanding of:
- How Active Directory authentication works in enterprise environments
- The relationship between DNS and Active Directory
- Why domain controllers require specific network configurations
- Best practices for domain architecture and naming conventions
- Common help desk scenarios involving domain accounts
